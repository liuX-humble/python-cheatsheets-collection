    <h3>XPath - Universal Search Language</h3>
    <pre><code><span class="comment"># XPath syntax</span>
    <span class="comment"># //tagName[@AttributeName="Value"]</span>
    <span class="comment"># Examples:</span>
    <span class="comment"># //article[@class="main-article"] - find article with class</span>
    <span class="comment"># //h1/text() - get h1 text</span>
    <span class="comment"># //div[@class="content"]//p - all p inside div</span>
    <span class="comment"># Navigation axes:</span>
    <span class="comment"># / - child elements</span>
    <span class="comment"># // - elements at any level</span>
    <span class="comment"># . - current element</span>
    <span class="comment"># .. - parent element</span>
    <span class="comment"># @ - attribute</span>
    <span class="comment"># Functions:</span>

    <span class="comment"># text() - text content</span>
